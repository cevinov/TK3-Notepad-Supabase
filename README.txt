Develop Mobile App (TK3) with Expo + Supabase (Relational Database)

Create note app

Node JS: v18.19.1 (LTS)

Initialize: npx create-expo-app@latest notepad

NPM Dependency: 
- npx expo install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar @react-navigation/native @expo/vector-icons @supabase/supabase-js

Run Project: npx expo start -c

Build APK: https://docs.expo.dev/build-reference/apk/
npm install --global expo-cli eas-cli
